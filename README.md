# Web Development Projects 🌐

Full-stack web applications from frontend to backend + APIs.

## Tech Stack
- Frontend: React, HTML/CSS/JS, Tailwind
- Backend: Node.js/Express, Next.js
- Database: MongoDB, PostgreSQL
- Deployment: Vercel, AWS Amplify

## Projects

### 1. Beginner: Task Dashboard with Login 🔐
Full-stack app with authentication.
- **Files**: `server.js`, `index.html`, `style.css`

### 2. Intermediate: API Integration Dashboard 📊
Weather + News dashboard with charts.
- **Files**: `dashboard.js`

### 3. Beginner: Portfolio Website 🖼️
Responsive personal portfolio.
- **Files**: `portfolio.html`

### 4. Intermediate: Full-Stack Todo App with React 🚀
MERN stack todo application.
- **Files**: React components (starter)

### 5. Bonus: Real-time Chat App 💬
Socket.io chat application.

## Quick Start
```bash
# Backend
npm init -y &amp;&amp; npm i express cors
node server.js

# Open frontend
# Uses Live Server extension or python -m http.server
```
